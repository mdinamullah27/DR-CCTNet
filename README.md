## ðŸŽ¯ Overview

This work proposes **DR-CCTNet**, a Modified Compact Convolutional Transformer designed to:

- Classify diabetic retinopathy from retinal fundus images
- Operate efficiently on **low-resolution inputs**
- Reduce computational complexity and training time
- Maintain strong classification accuracy
